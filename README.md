Smart Path is an AI-powered, voice-controlled smart mobility system for visually impaired people.

- Voice-based control
- AI obstacle detection
- Live traffic aware routing
- Smart navigation
- Emergency SOS
- Cross-platform support

## Tech Stack

- Python
- FastAPI
- OpenCV
- YOLOv8
- Google Maps API
- Speech Recognition
- Flutter (Mobile App)
